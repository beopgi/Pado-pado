## 개발 배경 및 필요성
- 현재 국내 해양 관광 및 라이프스타일 서비스는 항해 기록을 단순 저장하거나 아예 축적하지 못해, 데이터 기반 접근 및 활용이 제한적임. 이로 인해 사용자 경험이 축적되지 못해 고도화된 의사결정을 지원이 어려움. 따라서, 항해 로그를 분석하고 추천 및 안전 정보를 제공할 수 있는 AI 기반 보팅 라이프 플랫폼 구축이 필요함.

## 서비스 핵심 기능 
- AI 기반 해양 라이프스타일 추천 기능<br>사용자의 위치 정보 및 항해 로그 데이터, 공공데이터와 함께 해양 라이프스타일를 AI 기술을 통해 추천함
  > 해양 라이프스타일(=Coastal Lifestyle)은 바다(해양)라는 공간을 중심으로 여가·관광·스포츠·문화·소비·일상 습관이 형성되는 생활양식 전반을 뜻함
  
- 사용자의 항해정보 아카이브 및 공유<br>GPS 기반 위치 데이터를 활용하여 항해 경로, 이동 거리, 소요 시간 등을 자동으로 기록한다. 기록된 데이터는 서버에 저장하고, 사용자가 조회할 수 있도록 구현한다.
- 지도 기반 탐색 기능<br>지도 UI를 통해 주변 항해 포인트, 사용자 기록, 추천 장소 등을 시각적으로 확인할 수 있도록 한다. 지도 API와 연동하여 위치 기반 정보 제공이 가능하도록 구현한다.
- Product-Ready 수준의 어플리케이션 개발<br>
실제 해양 라이프스타일 관련한 제품 구매, 결제 서비스를 지원하도록 함.
- 크로스플랫폼 구현<br>
Flutter 기반으로 Android 및 iOS에서 동작 가능한 구조로 구현하며, 실제 배포를 고려한 앱 구조로 설계한다.
